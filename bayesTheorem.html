Bayes Theorem

  • P(H/E)
  • = [P(H) P(E|H)] / P(E) [1]

    a
  • = [P(H) P(E|H)] / [P(H)P(E|H) + P(~H)P(E|~H)]
  • P(H) = H/U (prior [2])
  • P(E) = E/U
  • P(E|H) = (E ∩ H) / H (likelihood [3])
  • P(H|E) = (E ∩ H) / E

  • [P(H) P(E|H)] / P(E)
    • = [H/U (H ∩ E) / H] / [E/U]
    • = H/U (H ∩ E)/L U/E
    • = H/U (H ∩ E)/H U/E
    • = (H ∩ E)/H
    • = P(H|E) (posterior [4])

  • P(H)P(E|H) + P(~H)P(E|~H)
    • = H/U (E ∩ H)/H + (U-H)/U (E - E ∩ H)/(U-H)
    • = H/U (E ∩ H)/H + (U-H)/U (E - E ∩ H)/(U-H)
    • = (E ∩ H)/U + (E - E ∩ H)/U
    • = E/U
    • =P(E)

Application:[5]

  • What is probablity that hypothesis is true given evidence (ie. P(E|H))?
  • evaluate hypothesis given new evidence
