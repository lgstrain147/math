Sum of Squares [1]

∑ k=0->n k2
  • s(n) = ∑ k=0->n k2 = 0 + 1 + 4 + 9 + 16 + 25 + ... + n2
  • s(n) = 0,1,5,14,30,... for n=0,1,2,3,4,...
  • d(n) = s(n) - s(n-1) = 1,4,9,16,... for n=1,2,3,4,... so s(n) is not linear
  • d2(n) = d(n) - d(n-1) = 3,5,7,... for n=2,3,4,... so s(n) is not quadratic
  • d3(n) = d2(n) - d2(n-1) = 2,2,2,... for n=3,4,... so s(n) is cubic
  • therefore s(n) = an3 + bn2 + cn + d
  • s(1) = 1 = a + b + c + d
  • s(2) = 5 = 8a + 4b + 2c + d
  • s(3) = 14 = 27a + 9b + 3c + d
  • s(4) = 30 = 64a + 16b + 4c + d
  • ∴ s(n) = 1/3n3 + 1/2n2 + 1/6n [2]
  • ∴ s(n) = n(n+1)(2n+1)/6[3]
